## Usage Instructions
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/liz4208/AI-Skills.git
   cd AI-Skills
   ```
2. **Install Requirements**:
   Ensure that you have the necessary dependencies. Run:
   ```bash
   pip install -r requirements.txt
   ```
3. **Running a Skill**:
   To run a specific skill, execute:
   ```bash
   python skills/skill_name.py
   ```
4. **Running Tests**:
   To run the test cases, use:
   ```bash
   pytest tests/
   ```
